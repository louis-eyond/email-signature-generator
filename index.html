Setup Instructions

Paste into Settings

  1. Click Copy signature above.
  2. In new Outlook, click the gear icon (top right) to open Settings.
  3. Go to Account → Signatures.
  4. Click + New signature and give it a name.
  5. Click into the editor and press Ctrl + V to paste.
  6. Wait a few seconds for images to load.
  7. Set as default for new messages and replies, then Save.
If paste doesn't preserve formatting, try Outlook on the Web at outlook.office.com. Pasting there is more reliable, and the signature syncs to your desktop client automatically.

Edit the signature file directly

  1. Open Classic Outlook → File → Options → Mail → Signatures.
  2. Click New, give it a name, type any placeholder, click Save.
  3. Close Outlook completely.
  4. Press Win + R, paste %appdata%\Microsoft\Signatures, press Enter.
  5. Right-click the .htm file with your signature name → Open with → Notepad.
  6. Press Ctrl + A then Delete.
  7. Click Copy HTML source above, return to Notepad and press Ctrl + V.
  8. Save and close. Reopen Outlook, set the signature as default.

Outlook for Mac

  1. Click Copy signature above.
  2. In Outlook, go to Outlook → Settings → Signatures.
  3. Click + and name the signature.
  4. Click into the editor and press Cmd + V to paste.
  5. Set as default in the dropdowns at the bottom.

Gmail

  1. Click Copy signature above.
  2. In Gmail, click the gear icon → See all settings → General → scroll to Signature.
  3. Click + Create new and name it.
  4. Click into the editor and paste.
  5. Under Signature defaults, set for new emails and replies.
  6. Scroll to the bottom and click Save Changes.

Apple Mail

  1. Click Copy signature above.
  2. In Mail, go to Mail → Settings → Signatures.
  3. Select your account, click +, name it.
  4. Click into the editor and paste.
  5. Untick Always match my default message font if ticked.
  6. Set as the default for the account.
